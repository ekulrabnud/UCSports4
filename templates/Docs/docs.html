Overview

UCTV sports scheduler automates and enhances communication of live sporting events carried on United Center in-house tv. It is a webapp that provides the functionality needed to achieve the above. TV listing data is provided from an API serviced by TV Media. This is a subscription service which is invoiced on monthly basis to United Center. Data is downloaded from the API and is entered into a local database. It can then be requested and formated for email distribution, PDF and infocaster display on UCTV channel guide.

Software Design

Operation

Application has 4 tabs

  • Channel Guide

    • This a static display of our inhouse channel guide. No functionality. Any edits to this must be performed indirectly on database by a qualified technician.

  • Live Sports Schedule

    • Displays live sports schedule. Defaults to "today start 1:00am stop 11:00pm". User can pick any date and time frame 7 days in the future.
    Click to filter schedule by date and times selected.
    Click to open a email window formated as basic html that mirrors currently displayed sports schedule. See Email Production for more info. A prompt indicates success or failure.
    This formats a PDF doc that mirrors currently displayed sports schedule and exports it to \\ARNFPUDM01\Arena Share\Technical Operations\brady\UC Sports Menu. Any games with the word "Chicago" are highlighted. A prompt indicates success or failure
    This will update/change the text file (located at\\ARNFPUDM01\Arena Share\Technical Operations\Luke\uctv\uctvSportstxt.txt) that feeds the channel guide infocaster. The text file created will mirror displayed sports schedule.A prompt confirms "success" or "fail". Note "success" only indicates that the text file was created it does not verify that the infocaster is working and has seen the updated data. Visually verify by tuning to channel guide at 1.1. It usually takes a minute or 2 to see new data.
    Click this in order to edit current schedule i.e change channel name/numbers or remove event from schedule.
    Click this to add an event that does not exist in database. This is useful if editors at TVMedia screw up which has happened from time to time.
    Click this to save any edits.Any events selected for deletion in "x" column will be deleted.
    Click this to request 7 days of data from API and completely reload database. If you have made any updates they will be lost. The purpose is to allow a user to pull updated data from the API during sporting tournaments since team names are often not determined when the initial schedule is produced.

  • Channel Lineup

    • Static display of all the channels that we can currently add to our lineup whether they be Comcast or Bell Satellite. Editing functionality not implemented yet

  • Sports Lineup

    • Static display of channels that pull data for live sports. At some point will combine this with Channel Lineup for better functionality.

Sports Schedule Production workflow

Ishare PDF

Click "PDF" button in Live Sprts Schedule Tab which will produce pdf of visible schedule. Log into Ishare and follow link to "UCTV Guide". Click on "Document Library". Delete any old schedules if needed. Upload new schedule from '\\ARNFPUDM01\Arena Share\Technical Operations\brady\UC Sports Menu. (note this location is subject to change)

Email Production

"Reply all" to last sports schedule email. Click email button on app. This will open a window with contents of email. Select ALL. COPY and PASTE into outlook. Copy link address of Ishare pdf schedule and paste to text attibutes of "Ishare PDF". Change date in subject of heading. Change "From" box to "Technology" Highlight home games in yellow if needed.
